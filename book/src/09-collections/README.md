# Coleções

Esse capítulo é destinado a abordar as coleções em Rust, como já falamos sobre String e &str (Capítulo 6) e sobre Vetor (Capítulo 8), daremos continuidade agora trazendo o assuntos a seguir:

- Sequências:
  - Vec<T>;
  - VecDeque<T>;
  - LinkedList<T>;
  
- Mapas:
  - HashMap<K, V>;
  - BTreeMap<K, V>.

- Conjuntos:
  - HashSet<T>;
  - BTreeSet<T>.

- Outra:
  - BinaryHeap<T>.
