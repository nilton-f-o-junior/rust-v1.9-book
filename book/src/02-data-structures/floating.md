# f32 e f64: ponto flutuante (floating-point)

Números que possuem uma parte fracionária ou números reais.

`f32`

```rust
// f32 = -3.4e+38 to +3.4e+38
let flutuante: f32 =  179.76;
println!("{}", flutuante);
```

`f64`

```rust
// f64 = -1.8e+308 to +1.8e+308
let flutuante: f64 =  490.76;
println!("{}", flutuante);
```
