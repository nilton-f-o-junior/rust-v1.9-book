# u: inteiros não-assinados

São tipos de dados que só podem armazenar números inteiros positivos ou zero.

`u8`

```rust
// u8 = 0 a 255
let inteiro: u8 = 16;
println!("{}", inteiro);
```

`u16`

```rust
// u16 = 0 a 65_535
let inteiro: u16 = 32;
println!("{}", inteiro);
```

`u32`

```rust
// u32 = 0 a 4_294_967_295
let inteiro: u32 = 64;
println!("{}", inteiro);
```

`u64`

```rust
// u64 = 0 a 18_446_744_073_709_551_615
let inteiro: u64 = 128;
println!("{}", inteiro);
```

`u128`

```rust
// u128 = 0 a ...
let inteiro: u128 = 256;
println!("{}", inteiro);
```
