# u: inteiros não-assinados (unsigned integers)

Números inteiros que podem ser apenas positivos.

`u8`

```rust
// u8 = 0 - 255
let numeros_u8: u8 = 255;
println!("{}", numeros_u8);
```

`u16`

```rust
// u16 = 0 - 65.535
let numeros_u16: u16 = 255;
println!("{}", numeros_u16);
```

`u32`

```rust
// u32 = 0 - 4.294.967.295
let numeros_u32: u32 = 255;
println!("{}", numeros_u32);
```

`u64`

```rust
// u64 = 0 - 18.446.744.073.709.551.615
let numeros_u64: u64 = 255;
println!("{}", numeros_u64);
```

`u128`

```rust
// u128 = 0 - ...
let numeros_u128: u128 = 255;
println!("{}", numeros_u128);
```
