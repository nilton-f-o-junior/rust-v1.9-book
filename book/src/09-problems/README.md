# Problemas

Seja bem-vindo! Sei que não foi fácil chegar até aqui, porém ainda estamos no início da sua jornada. Diferente das seções de "Perguntas", este capítulo tem o objetivo de não só revisar o que foi aprendido até agora, mas desafiá-lo a testar os conhecimentos que adquiriu, de forma que você possa resolver questões mais complexas que vão exigir um pouco mais do seu tempo e do seu raciocínio lógico.

Os problemas foram divididos em três níveis de dificuldade. A cada nível, mais conhecimento teórico e, principalmente, lógico será necessário. Porém, você terá uma proposta de resposta para cada um deles. O objetivo aqui é fazer você usar um pouco de cada assunto que viu até agora.

O livro ainda está no seu início e teremos muitos conteúdos pela frente, além de material complementar que será adicionado.

Desde já, desejo um ótimo desafio!
