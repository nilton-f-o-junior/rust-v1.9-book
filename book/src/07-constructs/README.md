# Construtos

Este capítulo apresenta os blocos de construção fundamentais para a criação de tipos personalizados e a definição de seu comportamento.

- `struct` para agrupar dados relacionados;
- `enum` para representar valores que podem ser uma de várias possibilidades;
- `impl` para adicionar métodos aos seus tipos;
- `trait` para definir um comportamento compartilhado que múltiplos tipos podem implementar.

Você encontrará questões práticas para reforçar sua compreensão destes conceitos de programação fundamentais.

> É mais fácil do que parece!
