# Construtos

Este capítulo apresenta os blocos de construção fundamentais para a criação de tipos personalizados e a definição de seu comportamento.

Começaremos com as estruturas de dados: struct para agrupar dados relacionados em tipos coerentes, e enum para representar valores que podem ser uma de várias variantes. Em seguida, exploraremos o comportamento: blocos impl para adicionar métodos aos seus tipos, e trait para definir um comportamento compartilhado que múltiplos tipos podem implementar.

Você encontrará questões práticas para reforçar sua compreensão destes conceitos de programação fundamentais.
