# Controle de Fluxo

Este capítulo aborda as estruturas de fluxo de controle do Rust, começaremos com as declarações condicionais: `if`, `else`, `else-if`, depois abordaremos sobre `loops`, `while` e `for` para iterar sobre coleções. Por fim abordaremos sobre `match`, recurso de *pattern matching* que lida com condições complexas com precisão e segurança.

Cada seção inclui perguntas práticas para reforçar sua compreensão sobre os assuntos, como essa parte é um pouco mais complexa estarei separando as questões.
