# Operadores de Atribuição

Usados para atribuir valores a variáveis.

`+=`

```rust
let mut num: u8 = 10;

num += 1;
println!("10 += 1 = {}", num);
```

`-=`

```rust
let mut num: u8 = 10;

num -= 1;
println!("10 -= 1 = {}", num);
```

`*=`

```rust
let mut num: u8 = 10;

num *= 2;
println!("10 *= 2 = {}", num);
```

`/=`

```rust
let mut num: u8 = 20;

num /= 3;
println!("20 /= 3 = {}", num);
```

`%=`

```rust
let mut num: u8 = 6;

num %= 4;
println!("6 %= 4 = {}", num);
```
