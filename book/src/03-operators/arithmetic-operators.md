# Operadores Aritméticos

Usados para cálculos matemáticos básicos.

`+`

```rust
let adi = 1 + 1;
println!("Adição = {:?}", adi);
```

`-`

```rust
let sub = 2 - 2;
println!("Subtração = {:?}", sub);
```

`*`

```rust
let mul = 3 * 3;
println!("Multiplicação = {:?}", mul);
```

`/`

```rust
let div = 4 / 4;
println!("Divisão = {:?}", div);
```

`%`

```rust
let rest = 5 % 5;
println!("Resto Divisão = {:?}", rest);
```
