# Comparação

O ato de verificar se dois ou mais itens são iguais, diferentes ou ordenados em relação um ao outro.

`== (igual a)`

```rust
let a = 1;
let b = 1;
println!("{}", a == b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a == b);
```

`!= (diferente de)`

```rust
let a = 1;
let b = 1;
println!("{}", a != b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a != b);
```

`> (maior que)`

```rust
let a = 1;
let b = 1;
println!("{}", a > b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a > b);
```

`< (menor que)`

```rust
let a = 1;
let b = 1;
println!("{}", a < b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a < b);
```

`>= (maior ou igual)`

```rust
let a = 1;
let b = 1;
println!("{}", a >= b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a >= b);
```

`<= (menor ou igual)`

```rust
let a = 1;
let b = 1;
println!("{}", a <= b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a <= b);
```
