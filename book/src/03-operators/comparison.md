# Comparação

O ato de verificar se dois ou mais itens são iguais, diferentes ou ordenados em relação um ao outro.

`==`

```rust
// igual
let a = 1;
let b = 1;
println!("{}", a == b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a == b);
```

`!=`

```rust
// diferente
let a = 1;
let b = 1;
println!("{}", a != b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a != b);
```

`>`

```rust
// maior que
let a = 1;
let b = 1;
println!("{}", a > b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a > b);
```

`<`

```rust
// menor que
let a = 1;
let b = 1;
println!("{}", a < b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a < b);
```

`>=`

```rust
// maior ou igual
let a = 1;
let b = 1;
println!("{}", a >= b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a >= b);
```

`<=`

```rust
// menor ou igual
let a = 1;
let b = 1;
println!("{}", a <= b);
```

```rust
let a = 2;
let b = 3;
println!("{}", a <= b);
```
