# Install Rust

To install on macOS, Linux, or other Unix-like operating system, run the following in your terminal and follow the on-screen instructions.

```md
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Be sure to visit the official Rust website to better understand the installation process and other tools.

```md
https://www.rust-lang.org/tools/install
```

Other Installation Methods" if you are on Windows.
