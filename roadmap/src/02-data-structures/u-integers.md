# u: unsigned integers

Whole numbers that can only be positive.

`u8`

```rust
// u8 = 0 - 255
let number_u8: u8 = 255;
println!("{}", number_u8);
```

`u16`

```rust
// u16 = 0 - 65.535
let number_u16: u16 = 255;
println!("{}", number_u16);
```

`u32`

```rust
// u32 = 0 - 4.294.967.295
let number_u32: u32 = 255;
println!("{}", number_u32);
```

`u64`

```rust
// u64 = 0 - 18.446.744.073.709.551.615
let number_u64: u64 = 255;
println!("{}", number_u64);
```

`u128`

```rust
// u128 = 0 - ...
let number_u128: u128 = 255;
println!("{}", number_u128);
```
