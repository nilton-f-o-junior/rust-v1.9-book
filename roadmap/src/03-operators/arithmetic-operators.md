# Arithmetic Operators

Used for basic mathematical calculations.

`+`

```rust
//
let sum = 1 + 1;
println!("Sum = {:?}", sum);
```

`-`

```rust
//
let sub = 2 - 2;
println!("Sub = {:?}", sub);
```

`*`

```rust
//
let mul = 3 * 3;
println!("Mul = {:?}", mul);
```

`/`

```rust
//
let div = 4 / 4;
println!("Div = {:?}", div);
```

`%`

```rust
// remainder: %
let rem = 5 % 5;
println!("Rem = {:?}", rem);
```
